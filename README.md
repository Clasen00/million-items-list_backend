# Список миллионов элементов

## Зависимости
- Node.js
- Express.js
- Redis

## Архитектура

Представляет собой REST API, которое позволяет выбирать, сортировать и фильтровать элементы списка.

## Структура проекта

Проект состоит из следующих компонентов:
- `server.ts`: основной файл сервера
- `src`: папка с исходным кодом
- `routes`: папка с роутами
- `models`: папка с моделями данных
- `controllers`: папка с контроллерами
- `middlewares`: папка с middleware
- `utils`: папка с утилитами
- `config`: папка с конфигурационными файлами
- `tests`: папка с тестами
